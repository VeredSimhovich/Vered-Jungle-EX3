# Vered Simhovich - Jungle Project 🦁

A responsive, interactive client-side web application simulating a virtual jungle. This project is submitted as Home Assignment #3.

## 🔗 Live Demo
Check out the live website here: [https://veredsimhovich.github.io/Vered-Jungle-EX3/](https://veredsimhovich.github.io/Vered-Jungle-EX3/)

## 🚀 Features
* **Interactive Animals:** Click on any animal card to hear its unique real-world sound.
* **Keyboard Support:** Trigger animal sounds by pressing the corresponding keys (`L`, `M`, `E`, `S`, `P`, `T`, `F`) on your keyboard.
* **Ambient Background:** Immersive jungle background music plays automatically upon page interaction.
* **Fully Responsive:** Developed using Flexbox layout architecture to ensure compatibility across all screen sizes (Desktop, Tablet, and Mobile).
* **Audio Optimization:** Built-in mechanism to stop the currently playing animal sound whenever a new one is selected to avoid audio overlap.

## 🛠️ Technologies Used
* HTML5
* CSS3 (Flexbox Layout)
* JavaScript (ES6, Event Listeners)
* Web Speech API (Advanced Text-to-Speech integration on right-click)
